# 蹬蹬公益站导航

蹬蹬公益站导航用于整理、推荐和宣传公开可访问的公益站与免费模型入口。

项目希望帮用户更快找到可用站点，也帮认真维护公益站的站长获得更多曝光。这里不会评价谁“最好”，而是尽量把每个站点的可用模型、额度方式、优势、限制和注意事项写清楚，方便用户自己判断。

## 导航内容

- 公益中转站
- 免费模型入口
- 官方免费推理平台
- 签到给额度的站点
- 注册或登录给临时额度的站点
- 限时开放、限时注册、短期福利站点
- 支持生图、Claude、DeepSeek、国内模型等不同方向的入口

## 收录信息

每个站点会尽量整理这些内容：

- 站点名称
- 访问链接
- 一句话总结
- 可用模型或主要能力
- 额度获取方式
- 主要优点
- 最大缺点
- 是否签到
- 是否生图
- 是否稳定
- 是否限时开放
- 是否偏一次性体验

公益站和免费模型入口的可用性、额度、模型列表、注册状态都可能变化。使用前请自行判断风险，注意账号、密钥和个人数据安全。
